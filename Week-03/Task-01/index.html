Voice Alignment — Week 2

"Plain, technical, direct, honest, no fluff."

  • Zero artificial gradients. White space makes metrics readable.
  • Monospace numbers. Columns and precision scores align correctly.
  • Accent = data highlight. Teal marks results, not decoration.
  • Quiet frame. The case studies are the loudest thing on the page.

03 / Style Note

Paste Into Your AI Workspace

Copy this two-line style note into your Claude Project so every future build task stays pixel-consistent.

Workspace Configuration Note
Fonts: Geist Mono (Headings, Code, Metrics — monospace, weight 600–700) and Geist (Body, Descriptions — sans-serif, weight 400–600). Colors: Near-Black #09090b (text), Near-White #fafafa (background), Accent Teal #0f766e (links, highlights, active states), Muted Zinc #71717a (subtext, borders, labels).
Mood: Technical, clear, and deliberately quiet — let the model accuracy figures, code outputs, and precision metrics be the loudest thing on every page. No decorative gradients. No stock imagery. Use real screenshots and real numbers.
